# 农田直播监控系统

基于 Vue3 + Vant UI 开发的移动端农产品直播平台，支持实时监控、用户互动和农场信息展示。

## 项目简介

一个高品质农产品直播展示平台，通过实时视频直播展示农产品生产环境，提供用户互动功能，助力农产品销售和品牌建设。

## 功能特性

#### 页面布局
- [x] 响应式设计，移动端优先

- [x] 现代简洁UI风格，主色调#149D56

- [x] 2x2网格布局展示四个直播画面

## 开发指南

### 环境要求
- Node.js 16+
- npm 8+

### 安装和运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
